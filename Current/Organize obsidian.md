1. Category (1 μόνο): πού ανήκει λειτουργικά (Projects/Areas/Resources/Archive
2. tags: τι είναι / θεματικές / τεχνολογίες / status
3. Κανόνας που δουλεύει: 
     - υποφάκελοι για “πού ανήκει” (Tech/Domain/Personal), 
     - tags για “τι είναι” (type/how-to, topic/\*, tech/\*).
4. κανόνας για να βρίσκεις “resources για issues” γρήγορα:
      - Αν ένα note έχει \#issue είναι το ίδιο issue note.
      - Αν έχει \#issue/support είναι supporting material και πρέπει να linkάρει προς ένα συγκεκριμένο issue note (1 link στην κορυφή αρκεί).
