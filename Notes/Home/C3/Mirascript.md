Extract the "Mirascript" archive on your key and keep only the "RT6_USERCOM.CMD" file from the "CMD" folder.  
Connect the USB key to the RT6 and answer "YES" to the script execution request.  
The script will ask to launch the "WRITE" command to which you will answer "NO".  
The script will then ask to launch the "READ" command to which you will answer "YES".  
Wait for the script to finish executing before unplugging the US key.  
On your PC, make a copy of the "User_config.dat" file on the USB key.  
Extract the archive of "DATEXPLORER".  
Launch "DATEXPLORER" and click on "Load Config", select the "RT6_2xx-user_config.dat" file.  
Click on "Load Dat" and select the "User_config.dat" file at the root of your USB key  
Navigate to the section indicated above and make changes indicated.  
Click on "Save" and remove the USB key from your computer.  
Connect the USB key to the RT6 which will ask you to execute the "USERCOM" script, answer "YES".  
The script will ask to launch the "WRITE" command to which you will answer "YES".  
Wait for the RT6 to display a message asking to reboot. ESPECIALLY DO NOT ANSWER YES. It will be necessary to press  
the eject button for 10 seconds, until the RT6 restarts.
