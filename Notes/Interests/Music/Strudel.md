Σύμφωνα με τα docs, αν τρέξεις στο φάκελο με τα samples σου:  
Strudel
 
```
cd samples  
npx @strudel/sampler
```
   

ο sampler στήνει έναν HTTP server στο:
 
```
samples('http://localhost:5432/');  
n("\<0 1 2\>").s("swoop smash")
```
   

– και έτσι το Strudel “βλέπει” τα samples σου μέσω http://localhost:5432
