
Στο **Obsidian**, το Zettelkasten υλοποιείται μέσω τριών βασικών τύπων σημειώσεων:

1. Fleeting Notes (Προσωρινές Σημειώσεις)

- **Τι είναι:** Γρήγορες σκέψεις ή ιδέες που καταγράφεις "στο πόδι" για να μην τις ξεχάσεις.
- **Στο Obsidian:** Μπορείς να χρησιμοποιείς το Daily Notes core plugin για να τις συγκεντρώνεις.

2. Literature Notes (Σημειώσεις Βιβλιογραφίας)

- **Τι είναι:** Σημειώσεις από πράγματα που διαβάζεις, ακούς ή βλέπεις (βιβλία, άρθρα, podcasts).
- **Κανόνας:** Γράψε τις με **δικά σου λόγια** (paraphrasing), όχι απλό copy-paste. The Archive - Literature Notes

3. Permanent Notes (Μόνιμες Σημειώσεις)

- **Τι είναι:** Οι τελικές, "ατομικές" σημειώσεις. Κάθε σημείωση περιέχει **μόνο μία ιδέα** (Atomicity).
- **Διασύνδεση:** Κάθε νέα μόνιμη σημείωση πρέπει να συνδέεται με τουλάχιστον μία παλαιότερη χρησιμοποιώντας `[[Internal Links]]`. Obsidian Help: Internal Links

---

Πώς να το στήσεις στο Obsidian:

1. **Unique IDs:** Χρησιμοποίησε το Zettelkasten prefixer core plugin για να δίνεις αυτόματα μοναδικούς αριθμούς (π.χ. `202310271030`) στα αρχεία σου.
2. **MOCs (Map of Content):** Δημιούργησε "χάρτες" που λειτουργούν ως πίνακες περιεχομένων για συγκεκριμένα θέματα, συνδέοντας πολλές μόνιμες σημειώσεις μαζί.
3. **Graph View:** Χρησιμοποίησε το Graph View για να βλέπεις οπτικά πώς οι ιδέες σου σχηματίζουν δίκτυα (clusters).

Χρήσιμα Plugins:

- **Templater:** Για να δημιουργείς αυτόματα τη δομή (Tags, Links, Metadata) σε κάθε νέα σημείωση. Templater GitHub
- **Dataview:** Για να δημιουργείς λίστες με σημειώσεις που δεν έχουν συνδεθεί ακόμα (unlinked notes).
